# Chrona 快速入门指南

## 🎉 欢迎使用 Chrona

Chrona 是一个智能时间管理 macOS 应用，使用 AI 帮助你高效安排每日任务。

## 📋 前置要求

- macOS 13.0 或更高版本
- Xcode 15.0 或更高版本（仅开发时需要）
- Qwen API Key（从阿里云获取）

## 🚀 快速开始

### 1. 构建应用

```bash
# 进入项目目录
cd Chrona.macOS

# 运行快速启动脚本
./run.sh
```

或者使用 Xcode：
```bash
open Chrona.xcodeproj
# 按 ⌘R 运行
```

### 2. 首次配置

应用启动后：

1. **点击"设置"按钮**
2. **配置 Qwen API Key**
   - 在"Qwen API Key"输入框中粘贴你的 API Key
   - 点击"保存"
   - 确认显示"已配置"状态

3. **设置工作时间段**（可选）
   - 默认时间段：09:00-12:00, 14:00-18:00
   - 可以添加、删除或修改时间段
   - 支持多个不连续的工作时间段

4. **配置提醒设置**（可选）
   - 默认提前 5 分钟提醒
   - 可以开启/关闭结束前提醒

5. **授权通知权限**
   - 首次使用时会弹出通知权限请求
   - 点击"允许"以启用任务提醒功能

### 3. 开始使用

#### 添加任务

1. 在左侧的多行输入框中输入任务
2. 每行一个任务，例如：
   ```
   回复客户邮件
   准备下午的会议材料
   完成项目报告
   ```
3. 点击"添加任务"按钮

#### 生成今日计划

1. 确保已添加至少一个任务
2. 点击顶部工具栏的"生成今日计划"按钮
3. AI 会自动：
   - 根据工作时间段安排任务
   - 为每个任务分配合理的时间
   - 提供 2-3 条行动提示
   - 标记无法安排的任务（如果时间不够）

#### 标记完成

- 点击任务前的圆圈图标标记为完成
- 已完成的任务会显示打勾标记

#### 生成每日总结

1. 一天结束时，点击"生成今日总结"
2. AI 会生成简短的工作总结
3. 总结会显示在右侧面板底部

### 4. 菜单栏功能

关闭主窗口后，应用会继续在菜单栏运行：

- **点击菜单栏图标**查看：
  - 今日进度（完成数/总数）
  - 快速添加任务
  - 生成计划/总结
  - 打开主窗口
  - 设置和退出

## 💡 使用技巧

1. **多行输入**: 可以一次性输入多个任务，每行一个
2. **工作时间段**: 支持午休等不连续时间段
3. **菜单栏快捷操作**: 不需要打开主窗口也能快速添加任务
4. **系统通知**: 任务开始前会自动提醒你

## 📁 数据存储位置

- **设置**: UserDefaults
- **API Key**: macOS Keychain（安全存储）
- **任务和计划**: `~/Library/Application Support/Chrona/`
  - `tasks.json`: 任务列表
  - `plan-YYYY-MM-DD.json`: 每日计划
  - `summary-YYYY-MM-DD.json`: 每日总结

## 🔧 故障排除

### 无法生成计划

- 检查是否已配置 API Key
- 确保至少添加了一个任务
- 检查网络连接

### 没有收到通知

- 检查系统设置 > 通知 > Chrona
- 确保已授权通知权限
- 检查"勿扰模式"是否开启

### 构建失败

- 确保 Xcode 版本 >= 15.0
- 确保 macOS 版本 >= 13.0
- 尝试清理构建：`xcodebuild clean`
